How to Improve Detection Accuracy
💡
Good lighting is critical. Make sure the face is well-lit from the front. Shadows create false positives in texture analysis.
📐
Face angle matters. Straight-on frontal views give the most accurate results. Avoid angles greater than 45° for live analysis.
🔍
Higher resolution = better accuracy. Use a 1080p webcam or upload high-quality images. Blurry or compressed frames reduce detection quality significantly.
⚡
Use Auto Mode strategically. Set the auto interval to 15–30s for continuous monitoring to avoid API rate limits while still catching changes.
🧪
Test with known deepfakes. Upload known deepfake images to calibrate sensitivity for your use case before relying on it in high-stakes situations.
🎯
Sensitivity tuning. Use sensitivity 4–5 for high-security environments. Use 2–3 for general monitoring to reduce false positives.
What This System Detects
🤖
GAN-generated faces — Fully AI-synthesized faces (StyleGAN, DALL-E, Midjourney, Stable Diffusion, etc.)
🔄
Face swaps — Real-time or post-processed face replacement using DeepFaceLab, FaceSwap, Reface, etc.
🎭
Face reenactment — Puppeteering techniques that animate a target face using another person's expressions.
⚠
Limitations: This system uses AI vision analysis and may produce false positives with unusual lighting, heavy makeup, or low-quality video. Always treat uncertain results as requiring human review.
